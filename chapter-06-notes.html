Segment 1: Measurement After Recognition, Cost Model, Revaluation Model

Current Segment
In This Segment

อ่านตามลำดับนี้: เลือกนโยบายวัดมูลค่าหลังรับรู้ -> ใช้กับทั้ง class -> เข้าใจ fair value และ sufficient regularity -> แยก assessment กับ formal revaluation

แกนของ TAS 16 ในช่วงนี้คือ หลังจากกิจการรับรู้ Property, Plant and Equipment (PPE) ครั้งแรกแล้ว จะวัดมูลค่าต่อในงบการเงินอย่างไร คำตอบคือกิจการเลือกได้ระหว่าง cost model และ revaluation model

Core rule: after initial recognition, PPE is subsequently measured using either the cost model or the revaluation model

Cost model

Carrying amount = Cost - Accumulated depreciation - Accumulated impairment losses

Cost model เป็น baseline model ของ TAS 16 และต้องจำให้แม่นว่ามันไม่ได้แปลว่า “ถือไว้ที่ต้นทุนเฉย ๆ” แต่เป็น cost net of accumulated depreciation และ accumulated impairment losses

Revaluation model

Carrying amount = Fair value at revaluation date - Subsequent accumulated depreciation - Subsequent accumulated impairment losses

Revaluation model ใช้ได้เมื่อ fair value can be measured reliably และเมื่อ revalue แล้ว carrying amount จะถูก reset ไปที่ fair value ก่อนจะเดินหน้าคิด depreciation และ impairment ต่อจากฐานใหม่

Policy Guardrails

  • การเลือก cost model หรือ revaluation model เป็นทางเลือกของกิจการ ไม่ใช่ข้อบังคับว่าทุกกิจการต้องตีราคาใหม่
  • ถ้าเลือก revaluation model ต้องใช้กับ entire class of PPE ไม่ใช่เลือกเฉพาะสินทรัพย์บางชิ้นที่ทำให้งบดูดี
  • หลัก entire class ช่วยลดโอกาส earnings management และทำให้สินทรัพย์ประเภทเดียวกันถูกวัดด้วยฐานที่สม่ำเสมอ
  • แม้ revalue แล้ว ก็ยังต้องคิด depreciation และติดตาม impairment ต่อ เพราะ fair value ในวันตีราคาเป็นเพียงฐานใหม่ ไม่ใช่จุดสิ้นสุดของการวัดมูลค่า

Fair Value และ Sufficient Regularity

เมื่อกิจการเลือก revaluation model มาตรฐานต้องการให้ทำ revaluation ด้วย sufficient regularity เพื่อไม่ให้ carrying amount ณ วันสิ้นงวดแตกต่างจาก fair value อย่างมีสาระสำคัญ

คำว่า fair value ที่ใช้ใน slide เชื่อมกับ TFRS 13 คือ price that would be received to sell an asset in an orderly transaction between market participants at the measurement date

Sufficient regularity = revalue often enough so carrying amount is not materially different from fair value at reporting date
สถานการณ์ของราคาตลาดความถี่ที่ควรคิดข้อสอบชอบหลอก
ราคาผันผวนมากต้องประเมินและอาจต้องตีราคาใหม่บ่อยมาก เช่น ทุกปีห้ามยึดตัวเลข 3 หรือ 5 ปีแบบแข็งตัว
ราคาค่อนข้างนิ่ง ค่อย ๆ ขยับอาจตีราคาใหม่ทุก 3 ปี หรือ 5 ปีได้ ถ้ายังไม่ทำให้งบคลาดเคลื่อนอย่างมีสาระสำคัญ3 ปี หรือ 5 ปี เป็นแนวทาง ไม่ใช่กฎตายตัวของ TAS 16
ราคาดูเหมือนไม่เปลี่ยนหลายปีอาจไม่ต้องทำ formal revaluation ใหม่ หาก assessment รองรับว่า fair value ไม่เปลี่ยนอย่างมีสาระสำคัญยังต้อง assess ทุกสิ้นงวด และต้องมีหลักฐานให้ผู้สอบบัญชีเชื่อได้

Assessment vs Formal Revaluation

Assessment

Assessment คือการประเมินเบื้องต้นทุกสิ้นงวดว่าราคาในบัญชีต่างจากราคาตลาดอย่างมีสาระสำคัญหรือไม่ ขั้นนี้ละเลยไม่ได้ แม้กิจการคิดว่าราคาสินทรัพย์นิ่งก็ตาม

Formal Revaluation

Formal revaluation คือการตีราคาใหม่อย่างเป็นทางการ เช่น จ้างผู้เชี่ยวชาญหรือจัดทำรายงานประเมินราคา แล้วปรับตัวเลขในงบการเงิน ขั้นนี้จำเป็นเมื่อ carrying amount ต่างจาก fair value อย่างมีสาระสำคัญ

Trap: ไม่ตีราคาใหม่หลายปี

กิจการอาจไม่ทำ formal revaluation ใหม่เป็นเวลา 5 ปีได้ ถ้า assess ทุกปี แล้วพิสูจน์ได้ว่า fair value ไม่ต่างจาก carrying amount อย่างมีสาระสำคัญ จึงไม่ต้องจ้างผู้ประเมินใหม่หรือบันทึกปรับปรุงราคาในปีนั้น

แต่กิจการจะอ้างว่าไม่ต้องประเมินอะไรเลยตลอด 5 ปีไม่ได้ และการทิ้งช่วงยาวถึง 10 ปี แม้มาตรฐานไม่ได้เขียนตัวเลขห้ามไว้ตรง ๆ ก็ยากมากในทางปฏิบัติ เพราะต้องมีหลักฐานรองรับว่าราคาสินทรัพย์ไม่ต่างอย่างมีสาระสำคัญทุกปี

Decision Flow ในข้อสอบ

  1. ถามก่อนว่ากิจการใช้นโยบาย cost model หรือ revaluation model
  2. ถ้าเป็น revaluation model ให้ถามต่อว่าการเลือกนี้ใช้กับ ทั้ง class หรือไม่
  3. ถ้าโจทย์ให้ fair value มา อย่ารีบใช้ทันที ต้องเช็กก่อนว่ากิจการอยู่ภายใต้ revaluation model หรือยัง
  4. ถ้าโจทย์ถามเรื่องรอบการตีราคา ให้แยก assessment ทุกสิ้นงวด ออกจาก formal revaluation ที่ทำเมื่อ carrying amount ต่างจาก fair value อย่างมีสาระสำคัญ
  5. ถ้า revalue แล้ว ให้คิดต่อเสมอว่าฐาน depreciation และประเด็น impairment หลังจากนั้นจะเปลี่ยนอย่างไร

Exam Watch

  • คำว่า cost model ไม่ได้แปลว่าไม่หัก impairment
  • revaluation model ตีราคาแล้วก็ยังต้องคิด depreciation และ impairment ต่อ
  • policy choice ใช้กับทั้ง class ไม่ใช่เฉพาะบางรายการ
  • การมี fair value ในโจทย์ไม่ได้แปลว่าต้องใช้เสมอ ต้องดูนโยบายบัญชีก่อน
  • ตัวเลข 3 ปี หรือ 5 ปี เป็นแนวทาง ไม่ใช่ใบอนุญาตให้ไม่ assess fair value ทุกปี
  • ถ้ากิจการบอกว่าไม่ต้อง formal revaluation หลายปี ต้องมีหลักฐานว่าราคาไม่ต่างจาก carrying amount อย่างมีสาระสำคัญ

Mini Test 1

Mini Test 1: Cost Model vs Revaluation Model

บริษัท เหนือฟ้าอุตสาหกรรม จำกัด ซื้อเครื่องจักร A วันที่ 1 มกราคม 20X1 ในราคา 900,000 บาท อายุการใช้งาน 6 ปี มูลค่าคงเหลือ 60,000 บาท และใช้วิธีเส้นตรง ณ 31 ธันวาคม 20X2 ผู้ประเมินรายงานว่า fair value ของเครื่องจักร A เท่ากับ 720,000 บาท ฝ่ายการเงินเสนอให้ตีราคาใหม่เฉพาะเครื่องจักร A เพราะมูลค่าสูงขึ้นชัดเจน ขณะที่กิจการยังมีเครื่องจักร B และ C ซึ่งอยู่ใน class เดียวกันและยังไม่ได้รับการประเมินราคาใหม่

  1. คำนวณค่าเสื่อมต่อปีและ carrying amount ของเครื่องจักร A ณ 31 ธันวาคม 20X2 ภายใต้ cost model
  2. ถ้ากิจการใช้ revaluation model สำหรับ class ของเครื่องจักรทั้งหมด จงระบุ carrying amount ใหม่ของเครื่องจักร A
  3. อธิบายว่าข้อเสนอให้ตีราคาใหม่เฉพาะเครื่องจักร A ถูกต้องหรือไม่ และเพราะเหตุใด
  4. ถ้าหลัง revaluation เครื่องจักร A เหลืออายุใช้งานอีก 4 ปี และ residual value ไม่เปลี่ยน จงคำนวณค่าเสื่อมปี 20X3
  5. อธิบายว่าทำไมการมี fair value ในโจทย์ไม่ได้หมายความว่ากิจการต้องใช้ revaluation model เสมอ
  6. ถ้ากิจการใช้ revaluation model แต่ไม่ได้ทำ formal revaluation ใหม่เป็นเวลา 5 ปี จะถือว่าผิด TAS 16 เสมอหรือไม่ อธิบายโดยแยก assessment กับ formal revaluation

Segment 2: Revaluation Increase, Revaluation Decrease, Sequence Effect

Current Segment
In This Segment

ช่วงนี้ลงลึกเรื่อง recognition location ของผลต่างจากการตีราคาใหม่: เมื่อไรเข้า OCI, เมื่อไรเข้า P/L, และทำไมต้องตามประวัติของ same asset ทุกครั้ง

เมื่อกิจการเลือกใช้ revaluation model คำถามใหญ่ต่อมาคือ “ส่วนต่างจากการตีราคาใหม่จะไปรับรู้ที่ไหน” จุดนี้เป็นแกนยากที่สุดของ Chapter 6 เพราะคำตอบไม่ได้เข้า profit or loss ทั้งหมด และไม่ได้เข้า OCI ทั้งหมดเสมอไป

Memory hook: reverse where it came from

Revaluation increase

หลักทั่วไป: Increase -> OCI -> Revaluation surplus

ถ้า carrying amount เพิ่มขึ้นเพราะ revaluation หลักทั่วไปคือรับรู้กำไรใน other comprehensive income (OCI) และสะสมไว้ใน equity ภายใต้ชื่อ revaluation surplus

เดิม carrying amount 40,000
fair value ใหม่ 45,000
increase 5,000

Dr Land 5,000
   Cr Revaluation Surplus 5,000

แต่มีข้อยกเว้นสำคัญ: ถ้าในอดีตสินทรัพย์ตัวเดียวกันเคยมี revaluation decrease ที่รับรู้ใน P/L มาก่อน increase รอบใหม่ต้อง reverse prior loss ใน P/L ก่อน ส่วนที่เกินจึงค่อยไป OCI

ตรรกะของมาตรฐานคือ ถ้าเคยรับรู้ downside ผ่าน P/L ไปแล้ว เมื่อมูลค่าฟื้นกลับขึ้นมาในส่วนเดียวกัน ก็ควร reverse ผ่าน P/L ก่อน เพื่อให้ recognition location สอดคล้องกับประวัติเดิมของสินทรัพย์นั้น

Revaluation decrease

หลักทั่วไป: Decrease -> Profit or loss

ถ้า carrying amount ลดลงเพราะ revaluation หลักทั่วไปคือรับรู้ขาดทุนใน P/L แต่ถ้ามี revaluation surplus ของสินทรัพย์ตัวเดียวกันคงอยู่ ให้ใช้ surplus เดิมก่อน ส่วนที่เกินค่อยลง P/L

เดิม carrying amount 40,000
fair value ใหม่ 38,000
decrease 2,000

Dr Loss from Revaluation 2,000
   Cr Land 2,000

ในทางกลับกัน หากสินทรัพย์ตัวนั้นมี revaluation surplus ค้างอยู่ การลดลงครั้งใหม่จะไม่วิ่งเข้า P/L ทั้งหมดทันที แต่ต้องใช้ยอด surplus เดิมตัดก่อน มาตรฐานมองว่านี่คือการดึง unrealized gain เดิมกลับลงมาก่อน

Sequence effect ของการตีราคาหลายรอบ

รูปแบบวิธีคิด
ขึ้นก่อน แล้วลงทีหลังใช้ revaluation surplus เดิมก่อน แล้วส่วนเกินค่อยลง P/L
ลงก่อน แล้วขึ้นทีหลังreverse prior loss ใน P/L ก่อน แล้วส่วนเกินค่อยไป OCI

Decision Flow สำหรับข้อสอบหลายปี

  1. หา carrying amount ก่อน revaluation ของปีนั้นให้ถูกก่อน
  2. เทียบกับ fair value ใหม่เพื่อหา increase หรือ decrease
  3. เปิดประวัติเดิมของ same asset ว่าเคยมี prior loss ใน P/L หรือ prior surplus ใน OCI คงเหลือเท่าไร
  4. จัดสรรผลต่างไปยัง P/L และ OCI ตามประวัติเดิม ไม่ใช่อาศัยความจำแบบเหมารวมว่า increase ต้องไป OCI เสมอ

Exam Watch

  • Increase ไม่ได้เข้า OCI ทั้งหมดเสมอไป
  • Decrease ไม่ได้เข้า P/L ทั้งหมดเสมอไป
  • คำว่า same asset สำคัญมาก ห้ามเอาประวัติคนละสินทรัพย์มาหักกลบ
  • ต้องแยกให้ออกระหว่าง measurement amount กับ recognition location เพราะบางคนคิด fair value ถูกแต่ตอบผิดว่าต้องไปที่ P/L หรือ OCI

Mini Test 2

Mini Test 2: Revaluation Increase / Decrease

บริษัท ศิลาไทย จำกัด มีที่ดินแปลงหนึ่งซึ่งรับรู้ครั้งแรกวันที่ 1 มกราคม 20X1 ในราคาทุน 1,500,000 บาท และกิจการใช้นโยบาย revaluation model สำหรับ class ของที่ดินทั้งหมด โดยมี fair value ณ สิ้นปี 20X2 = 1,680,000 บาท, สิ้นปี 20X3 = 1,420,000 บาท, สิ้นปี 20X4 = 1,560,000 บาท และสิ้นปี 20X5 = 1,740,000 บาท

  1. จัดทำตารางวิเคราะห์ carrying amount หลัง revaluation ของแต่ละปี พร้อมแยกจำนวนที่เข้า OCI และ P/L
  2. จัดทำ journal entries ของแต่ละปี
  3. อธิบาย logic ของปี 20X4 และ 20X5 ว่าทำไม recognition location จึงไม่เหมือนกัน
  4. ระบุให้ชัดว่าในแต่ละปีมี revaluation surplus คงเหลือปลายปีเท่าไร

Segment 3: Accumulated Depreciation at Revaluation Date, Eliminate Method, Restate Method

Current Segment
In This Segment

ช่วงนี้เน้น mechanics ที่ออกสอบบ่อยมากเมื่อสินทรัพย์เสื่อมราคาได้ถูก revalue: การจัดการ accumulated depreciation, ความต่างของ eliminate method กับ restate method, และเหตุผลที่ทั้งสองวิธีต้องจบที่ net amount เดียวกัน

เมื่อสินทรัพย์ที่คิดค่าเสื่อมอยู่แล้วถูก revalue ประเด็นไม่ได้มีแค่ว่า fair value ใหม่เท่าไร แต่ต้องตอบด้วยว่า accumulated depreciation เดิมจะจัดการอย่างไร เป้าหมายสุดท้ายคือ net carrying amount หลัง revaluation ต้องเท่ากับ revalued amount

Core target: net carrying amount after revaluation = revalued amount

Worked Example Setup

  • Cost = 100,000
  • Useful life = 5 ปี
  • Accumulated depreciation ณ สิ้นปี 25X2 = 40,000
  • Carrying amount ก่อน revaluation = 60,000
  • Fair value ใหม่ = 72,000

Eliminate method

วิธีนี้ล้าง accumulated depreciation ออกก่อน แล้วค่อยปรับยอดสุทธิให้ถึง revalued amount

Step 1 ล้าง accumulated depreciation
Dr Accumulated Depreciation 40,000
   Cr Equipment 40,000

หลัง Step 1
Equipment 60,000
Accumulated depreciation 0

Step 2 ปรับจาก 60,000 เป็น 72,000
Dr Equipment 12,000
   Cr Revaluation Surplus 12,000

Restate method

วิธีนี้ไม่ล้าง accumulated depreciation ทั้งหมด แต่ restate ทั้ง gross amount และ accumulated depreciation ให้เป็นสัดส่วนใหม่

Ratio = 72,000 / 60,000 = 1.2

Gross amount ใหม่ = 100,000 x 1.2 = 120,000
Accumulated depreciation ใหม่ = 40,000 x 1.2 = 48,000

Dr Equipment 20,000
   Cr Accumulated Depreciation 8,000
   Cr Revaluation Surplus 12,000

ความต่างของสองวิธีอยู่ที่ mechanics ไม่ใช่ผลลัพธ์สุทธิ เพราะไม่ว่าวิธีใด สุดท้ายต้องจบที่ net carrying amount = 72,000

หลัง revaluation แล้ว งานบัญชียังไม่จบ เพราะ revalued amount ใหม่จะส่งผลต่อค่าเสื่อมในอนาคต การโอน revaluation surplus และการบันทึกเมื่อสินทรัพย์ถูกขาย

Depreciation after revaluation

Depreciation after revaluation = (Revalued carrying amount - Residual value) / Remaining useful life

ตัวอย่างเดิม: revalued amount = 72,000 และอายุคงเหลือ 3 ปี ดังนั้น depreciation ใหม่ต่อปี = 24,000 จุดสำคัญคือห้ามกลับไปใช้ต้นทุนเดิมเป็นฐานคิดค่าเสื่อม

เหตุผลเชิงตรรกะคือ เมื่อกิจการ reset carrying amount ไปแล้ว ค่าเสื่อมในอนาคตก็ต้องสะท้อนฐานใหม่ด้วย ไม่เช่นนั้นงบจะใช้มูลค่าหลัง revaluation แต่ยังตัดค่าเสื่อมจากฐานเดิม ซึ่งขัดกันเอง

Transfer of Revaluation Surplus

Revaluation surplus อยู่ใน equity และสามารถโอนไป retained earnings ได้ แต่ก่อนจำวิธีโอน ต้องเข้าใจก่อนว่าทำไมมาตรฐานจึงยอมให้โอนได้

Why Transfer Exists

ตามกรอบของ TAS 16 ส่วนเกินจากการตีราคาสินทรัพย์หรือ revaluation surplus เป็นกำไรที่ยังไม่เกิดขึ้นจริง (unrealized gain) จึงถูกเก็บไว้แยกในส่วนของเจ้าของ ไม่ไปรวมกับกำไรสะสมตั้งแต่แรก

แต่เมื่อเวลาผ่านไปจากการใช้งานสินทรัพย์ หรือเมื่อสินทรัพย์ถูกขายออกไป กำไรส่วนนี้จะค่อย ๆ มีลักษณะ “เกิดขึ้นจริงมากขึ้น” ในเชิงบัญชี มาตรฐานจึงอนุญาตให้ย้ายยอดดังกล่าวไปไว้ใน retained earnings เพื่อสะท้อนว่ายอดนี้พร้อมถูกพิจารณาใช้ในส่วนของเจ้าของมากขึ้น เช่น การพิจารณาปันผลตามนโยบายของกิจการ

มาตรฐานเปิดทางให้โอนได้ 2 วิธีหลัก

  1. โอนเมื่อสินทรัพย์ถูกจำหน่าย คือโอนยอดคงเหลือของ revaluation surplus ทั้งก้อนเมื่อขาย บริจาค เลิกใช้ หรือ derecognize สินทรัพย์นั้น
  2. โอนทยอยตามการใช้งานของสินทรัพย์ คือโอนบางส่วนทุกงวดตามผลของค่าเสื่อมราคาที่เพิ่มขึ้นหลังการตีราคาใหม่

วิธีที่นิยมใช้อธิบายในข้อสอบคือกรณีโอนทยอย เพราะสะท้อนว่า “ค่าเสื่อมที่เพิ่มขึ้นจากการตีราคาใหม่” ได้กดกำไรสุทธิของแต่ละงวดไปแล้ว และค่าเสื่อมส่วนเพิ่มนี้เปรียบเสมือนการใช้งานที่เกิดขึ้นจริงของ unrealized gain ที่เคยเก็บไว้ใน revaluation surplus จึงสามารถดึงยอดบางส่วนจาก revaluation surplus มาเติมกลับเข้า retained earnings ได้

Transfer amount = Depreciation (Revalued Amount) - Depreciation (Original Cost)
ถ้าค่าเสื่อมตามต้นทุนเดิม = 20,000
ค่าเสื่อมหลัง revaluation = 24,000
ส่วนต่างที่โอนได้ = 4,000

Dr Revaluation Surplus 4,000
   Cr Retained Earnings 4,000

รายการนี้เป็นเพียง equity to equity transfer ไม่มีผลกระทบต่อ profit or loss ของงวดนั้น และโดยสาระแล้วควรสะท้อนผ่าน statement of changes in equity โดยตรง ไม่ใช่วิ่งผ่านหน้า P&L

Statement Effect ที่ต้องแยกให้ออก

  • Depreciation expense กระทบกำไรขาดทุนของงวด
  • Gain or loss on disposal กระทบกำไรขาดทุนของงวด ณ วันที่ขาย
  • Transfer of revaluation surplus ไม่กระทบกำไรขาดทุน แต่ย้ายยอดภายในส่วนของเจ้าของ

Policy Watch

  • การโอน revaluation surplus เป็น ทางเลือกเชิงนโยบายบัญชี มาตรฐานอนุญาตให้ทำหรือไม่ทำก็ได้ แต่ถ้ากิจการเลือกแนวทางใดควรใช้อย่างสม่ำเสมอ
  • ห้ามบันทึกการโอนนี้ผ่าน profit or loss เด็ดขาด
  • ตอนทำข้อสอบต้องแยกให้ชัดระหว่าง depreciation expense, gain/loss on sale, และ transfer of revaluation surplus เพราะทั้งสามรายการมีผลคนละที่ในงบ

Disposal Sequence

Disposal sequence คือ “ลำดับขั้นตอนในการตัดจำหน่ายสินทรัพย์” ออกจากงบการเงินของกิจการ เหตุผลที่ต้องมีลำดับนี้ชัดเจนก็เพราะสินทรัพย์ระยะยาวหนึ่งรายการไม่ได้ผูกกับแค่บัญชีสินทรัพย์เพียงบรรทัดเดียว แต่ผูกกับหลายบัญชีพร้อมกัน เช่น

  • บัญชีสินทรัพย์: แสดงราคาทุนเดิมหรือ revalued amount
  • บัญชีค่าเสื่อมราคาสะสม: แสดงมูลค่าที่ถูกใช้ไปแล้ว
  • บัญชี revaluation surplus: แสดง unrealized gain ที่ยังค้างอยู่ใน equity ถ้ากิจการใช้ revaluation model

ดังนั้นเมื่อสินทรัพย์ออกจากกิจการ เราจะลบเฉพาะบัญชีสินทรัพย์ทิ้งไม่ได้ แต่ต้องเคลียร์ทุกบัญชีที่เกี่ยวข้องให้ครบ เพื่อให้ยอดของสินทรัพย์นั้นกลายเป็นศูนย์ทั้งหมดในงบการเงิน

Disposal sequence = update depreciation -> record disposal -> transfer remaining revaluation surplus
  1. Update depreciation: บันทึกค่าเสื่อมราคาจากต้นงวดถึงวันที่จำหน่ายจริงก่อน
  2. Record disposal: รับรู้เงินสด ล้างสินทรัพย์ ล้าง accumulated depreciation และบันทึก gain/loss on disposal ใน P/L
  3. Transfer surplus: โอน revaluation surplus ที่เกี่ยวข้องไป retained earnings โดยตรง

Worked example: disposal of a revalued asset

สมมติ ณ วันที่จะจำหน่ายเครื่องจักร กิจการมีข้อมูลดังนี้

รายการจำนวน
Machine (revalued amount)120,000
Accumulated depreciation ยกมาต้นปี30,000
Revaluation surplus คงเหลือใน OCI40,000
Cash received on disposal100,000
Depreciation for half-year before disposal15,000

Step 1: Update depreciation

ต้องบันทึกค่าเสื่อมราคาจากต้นปีถึงวันที่จำหน่ายจริงก่อน เพื่อให้ book value ณ วันขายถูกต้อง

Dr Depreciation Expense 15,000
   Cr Accumulated Depreciation 15,000

หลังรายการนี้ accumulated depreciation จะเพิ่มจาก 30,000 เป็น 45,000 บาท และ carrying amount ณ วันขายจะเท่ากับ 120,000 - 45,000 = 75,000 บาท

Step 2: Record disposal

เมื่อขายได้เงินสด 100,000 บาท เทียบกับ carrying amount 75,000 บาท จึงเกิด gain on disposal เท่ากับ 25,000 บาท ซึ่งต้องรับรู้ใน profit or loss

Dr Cash 100,000
Dr Accumulated Depreciation 45,000
   Cr Machine (Revalued Amount) 120,000
   Cr Gain on Disposal of Asset 25,000

Step 3: Transfer remaining surplus

เมื่อเครื่องจักรถูกขายไปแล้ว revaluation surplus ที่เคยค้างอยู่สำหรับสินทรัพย์นี้ถือว่าไม่ควรค้างอยู่ใน OCI อีกต่อไป จึงโอนทั้งหมดไป retained earnings

Dr Revaluation Surplus 40,000
   Cr Retained Earnings 40,000

รายการนี้ไม่ผ่าน P/L แต่เป็นการย้ายยอดภายในส่วนของเจ้าของโดยตรง

Key Takeaways

  • หลังจบ disposal sequence บัญชีของสินทรัพย์รายการนั้นควรถูกเคลียร์ครบ: asset = 0, accumulated depreciation = 0, และ revaluation surplus ของสินทรัพย์นั้น = 0
  • Gain or loss on disposal กับ transfer of revaluation surplus เป็นคนละเรื่อง: อย่างแรกผ่าน P/L แต่อย่างหลังโอนไป retained earnings โดยตรงและไม่ผ่าน P/L

Exam Watch

  • หลัง revaluation ค่าเสื่อมต้องคิดจากฐานใหม่
  • ตอนขายสินทรัพย์ ห้ามปน gain on sale กับ transfer of revaluation surplus เพราะเป็นคนละขั้นตอนและคนละผลกระทบในงบ

ข้อเสนอให้ตีราคาใหม่เฉพาะเครื่องจักร A ไม่ถูกต้อง เพราะ TAS 16 กำหนดให้ใช้นโยบาย revaluation model กับ entire class of PPE ไม่ใช่เลือกเฉพาะสินทรัพย์ที่ทำให้กิจการดูดี

หลัง revaluation และเมื่ออายุใช้งานคงเหลือ 4 ปี โดย residual value ยังเป็น 60,000 บาท ค่าเสื่อมปี 20X3 = (720,000 - 60,000) / 4 = 165,000 บาท

การมี fair value ในโจทย์ไม่ได้บังคับให้ใช้ revaluation model เสมอ เพราะ fair value อาจเป็นเพียงข้อมูลประกอบ แต่ recognition ในงบยังต้องขึ้นกับนโยบายบัญชีที่กิจการเลือกใช้

กรณีไม่ได้ทำ formal revaluation ใหม่เป็นเวลา 5 ปี ไม่ได้ผิด TAS 16 เสมอไป ถ้ากิจการ assess ทุกสิ้นงวด แล้วมีหลักฐานว่ามูลค่ายุติธรรมไม่ต่างจาก carrying amount อย่างมีสาระสำคัญ แต่ถ้าไม่มีการประเมินหรือไม่มีหลักฐานรองรับ การเว้นช่วงยาวจะมีปัญหากับหลัก sufficient regularity และผู้สอบบัญชีมักไม่ยอมรับ
